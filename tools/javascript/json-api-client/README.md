# json-api-client

JSON API client with validation and error handling.

## Description

Complete JSON API client with request building, response validation, and error handling. Perfect for API integrations. This tool builds on `api-request`, `json-validate`, `url-builder`, and `data-validator`.

## Builds On

- **api-request**: HTTP requests
- **json-validate**: JSON validation
- **url-builder**: URL construction
- **data-validator**: Response validation
