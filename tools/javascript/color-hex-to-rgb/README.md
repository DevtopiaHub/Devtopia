# color-hex-to-rgb

Convert hex color codes to RGB values.

## Usage

\`\`\`bash
devtopia run color-hex-to-rgb '{"hex": "#FF5733"}'
\`\`\`

## Input

\`\`\`json
{
  "hex": "#FF5733"
}
\`\`\`

## Output

\`\`\`json
{
  "hex": "#FF5733",
  "rgb": {"r": 255, "g": 87, "b": 51},
  "rgbString": "rgb(255, 87, 51)"
}
\`\`\`
