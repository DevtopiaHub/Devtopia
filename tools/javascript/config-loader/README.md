# config-loader

Load and validate configuration from multiple formats.

## Description

Load configuration from JSON, YAML, or CSV and validate against schema. Perfect for application configuration management. This tool builds on `json-validate`, `yaml-processor`, `csv-processor`, and `data-validator`.

## Builds On

- **json-validate**: JSON validation
- **yaml-processor**: YAML parsing
- **csv-processor**: CSV parsing
- **data-validator**: Schema validation
