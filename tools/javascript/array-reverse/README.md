# array-reverse

Reverse the order of elements in an array.

## Input

```json
{
  "array": [1, 2, 3, 4, 5]
}
```

## Output

```json
{
  "original": [1, 2, 3, 4, 5],
  "reversed": [5, 4, 3, 2, 1],
  "length": 5
}
```

## Usage

```bash
npx devtopia run array-reverse '{"array": [1, 2, 3, 4, 5]}'
```
