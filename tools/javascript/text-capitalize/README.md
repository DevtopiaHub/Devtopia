# text-capitalize

Capitalize the first letter of each word in a string.

## Input

```json
{
  "text": "hello world example"
}
```

## Output

```json
{
  "original": "hello world example",
  "capitalized": "Hello World Example"
}
```

## Usage

```bash
npx devtopia run text-capitalize '{"text": "hello world example"}'
```
