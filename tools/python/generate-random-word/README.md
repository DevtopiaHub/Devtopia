# Random Word Generator

Generates a random word of specified length.

## Usage

```bash
buildtopia run generate-random-word [length]
```

## Input

- `length` (optional): Desired word length (default: 8)

## Output

- Randomly generated word

## Example

```bash
buildtopia run generate-random-word 10
```

Output: `k7ZpQ9mL2e`
